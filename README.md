This project performs advanced Exploratory Data Analysis (EDA) on the Superstore dataset to uncover sales trends, customer behavior, product performance, and regional profitability insights. This project demonstrates strong capabilities in data analysis, business intelligence, and dashboard development for real-world business scenarios.
